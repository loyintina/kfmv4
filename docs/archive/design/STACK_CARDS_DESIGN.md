# 堆叠卡片 — 配色系统设计文档

> 右侧边缘左滑唤出的卡片堆叠面板，用于盒子的快速唤出与切换。
> 三套配色方案存档，当前活动方案：暮光 (Twilight)。

---

## 设计约束

卡片数量：7 张（对应 7 个盒子入口）
堆叠方向：从上到下，暖到冷光谱排列
交互手势：
  - 右边缘 30px 内左滑 -> 呼出面板
  - 垂直滑动 -> 切换聚焦卡片
  - 点击聚焦卡片 -> 打开对应盒子（未来）
  - 右滑 / 点击背景 -> 关闭面板

卡片三层视觉结构：
  - 聚焦卡片：边框高亮色，阴影更亮，内容完全可见，向右轻弹
  - 邻近卡片：边框半透明，阴影淡化，内容可见
  - 底层卡片：仅露边缘色条(2-3px)，不显示内容

---

## 方案一：暮光 Twilight（当前活动）

低饱和暖到冷过渡，像黄昏到夜晚的天色过渡。

色值：
  - Dusty Rose:    #D4899B  暖粉（最上，聚焦色）
  - Terracotta:    #D4A080  陶土
  - Antique Gold:  #C9B07A  古董金
  - Sage:          #8FB58F  鼠尾草绿
  - Dusty Teal:    #7DA8B8  灰蓝绿
  - Muted Lavender:#8E8EB8  薰衣草紫
  - Soft Violet:   #A08CC4  柔紫（最下）

设计特点：
  - 所有颜色都加了灰调（降低饱和度），避免纯色的张扬
  - 暖端偏暗红/橙黄，过渡自然
  - 冷端偏灰蓝/灰紫，保持统一的高级感
  - 中间鼠尾草绿作为冷暖分界，过渡平滑
  - 在深色背景下，毛玻璃效果叠加后色彩会进一步柔和

## 方案二：琉璃 Colored Glass

高明度低饱和，像彩色玻璃透光的效果。

色值：
  - Pale Rose:    #E8B4B4  浅玫瑰
  - Warm Peach:   #E8C8A0  暖桃
  - Soft Gold:    #D4CC94  柔金
  - Mint:         #A0C8A0  薄荷
  - Powder Blue:  #94B8D4  粉蓝
  - Lavender:     #A8A8D4  薰衣草
  - Lilac:        #C8A8D4  丁香紫

设计特点：
  - 全系高明度(~75-85%)，比暮光亮一个层级
  - 低饱和(~15-30%)，温和不刺眼
  - 适合需要更轻盈视觉感受的场景
  - 在毛玻璃叠加后呈现"透光"质感

---

## 方案三：星云 Nebula

中饱和暖到冷过渡，比暮光更浓郁。

色值：
  - Rose:        #B46478  玫瑰红
  - Amber:       #C88C5A  琥珀
  - Moss Gold:   #B4AA50  苔金
  - Teal Green:  #50A880  青绿
  - Steel Blue:  #5088C8  钢蓝
  - Purple:      #6C5CC8  紫
  - Violet:      #9650C8  紫罗兰

设计特点：
  - 饱和度更高，色彩更有存在感
  - 对比暮光更浓烈，适合需要强视觉层次时
  - 暖端偏浓郁，冷端偏深沉
  - 在深色背景上发光感更强

## 未来扩展

- 主题切换：通过 CSS 变量在三个配色间切换
- 卡片数量适应：根据屏幕高度自动计算堆叠偏移量
- 聚焦态深度：聚焦卡向右弹出距离可配置
