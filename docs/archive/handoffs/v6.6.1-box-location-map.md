### Box 位置映射设计

详见 `docs/design/BOX_LOCATION_MAP_SPEC.md`（已标记为 implemented）。

核心机制：
- `_boxLocationMap: Map<string, BoxLocation>` — 文件路径 → Box 屏幕坐标的反向索引
- `rebuildTree` 末尾自动重建
- `executeOnPath(path, 'tap')` — 绕过坐标直接触发文件行 tap
- AI 通过 `click-path` 命令操作，不再依赖合成 PointerEvent 坐标

### 审计发现

跨模块死代码审计发现：
- 无 GestureRegistry 绕过（唯一逃逸已被清理）
- 主要死代码来源：遗留替代实现（`container_AddRoot*`）、未使用的 API 设计（`anim.play()`/`kill()`体系）、未消费的导出（`*Capability`）
- `animation-registry.ts` 删除了命名动画管理 API（`play`/`kill`/`killAll`/`reverse`/`has`），保留 `scope`/`to`/`timeline`/`killTweensOf`——这些是实际被使用的
