# KFM v4 文档归档

按功能分类的历史文档。每份文件都有 status 标记。

**子目录导览：**

| 目录 | 内容 | 什么时候值得翻 |
|------|------|---------------|
| `handoffs/` | 版本交接记录 | 追溯某次改动的原始上下文时 |
| `design/` | 已完成的设计文档（Registry/卡片/引擎/愿景） | 改对应模块代码前翻阅 |
| `standards/` | 调试/Bug/测试/原则 | `BUG_AUDIT_REGISTRY.md` Ch.2 含额外诊断细节 |
| `bugs/` | Bug 根因分析 | 排查同类问题时参考 |
| `legacy/` | 已被替换的旧版本 | 基本不用翻 |

**状态说明**：`active`=仍有效建议阅读，`superseded`=已被新版覆盖，`completed`=计划已执行完。
